防抖（debounce）：就是指触发事件后在 n 秒内函数只能执行一次，如果在 n 秒内又触发了事件，则会重新计算函数执行时间。

例子：短时间多次点击按钮
```
<div>
  <input type="button" id="btn" value="" />
</div>
<script type="text/javascript">
const button = document.getElementById("btn");
const pay = function () {
  console.log(this);
  console.log("Have paid");
};
const debounce = function (fn, delay) {
  let timer;
  return function () {
    clearTimeout(timer);
    const args = arguments;
    const context = this;
    timer = setTimeout(() => {
    fn.call(context, ...args);
  }, delay);
  //timer是一个number类型
  console.log(timer);
  };
};
button.addEventListener("click", debounce(pay, 1000));
</script>
```

节流(throttle): 就是指连续触发事件但是在 n 秒中只执行一次函数，与防抖不同，执行时间是第一次触发事件

例子：窗口改变大小加一个事件。

计时器版：
```
let coloring = function () {
  let r = Math.floor(Math.random() * 255);
  let g = Math.floor(Math.random() * 255);
  let b = Math.floor(Math.random() * 255);
  document.body.style.backgroundColor = `rgb(${r},${g},${b})`;
};
 const throttle = function (fn, delay) {
  let timer;
  return () => {
    let context = this;
    let args = arguments;
    if (timer) {
      return;
    }
    timer = setTimeout(() => {
    fn.call(context, ...args);
    timer = undefined;
    }, delay);
  };
};
window.addEventListener("resize", throttle(coloring,1000));
```
时间戳版： 
```
const throttle2 = function (fn, delay) {
  let timestamp=0;
  return () => {
    let context = this;
    let args = arguments;
    let now=new Date();
    if (now-timestamp>delay) {
      fn.call(context,...args);
      timestamp=new Date();
    }
  };
};
```
